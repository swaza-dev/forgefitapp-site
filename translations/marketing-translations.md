# Marketing Copy Translations Template

This file contains the marketing copy that needs to be added to all language translation files.

## Structure

Add a "marketing" section to each translation JSON file with the following keys:
- marketing.intro.title
- marketing.intro.text
- marketing.aiCoach.title
- marketing.aiCoach.text
- marketing.accountability.title
- marketing.accountability.text
- marketing.personalization.title
- marketing.personalization.text
- marketing.analytics.title
- marketing.analytics.text
- marketing.nutrition.title
- marketing.nutrition.text
- marketing.features.title
- marketing.features.list (array)
- marketing.cta.text
